# 📊 Dashboard de Vendas - Excel

## 📌 Descrição do Projeto

Este projeto tem como objetivo a criação de um **dashboard de vendas** a partir de dados brutos, transformando-os em informações visuais claras e úteis.

O dashboard permite analisar o desempenho de vendas, identificar padrões e apoiar a tomada de decisões baseadas em dados.

---

## 🎯 Objetivos

* Organizar dados de vendas
* Criar visualizações claras e interativas
* Facilitar a análise de desempenho
* Apoiar decisões estratégicas

---

## 📂 Dados Utilizados

Os dados utilizados neste projeto incluem informações como:

* Tipo de assinatura (Subscription Type)
* Plano (Core, Standard, Ultimate)
* Valores de vendas
* Indicadores de desempenho

Os dados foram tratados e organizados para permitir a criação de tabelas dinâmicas e gráficos.

---

## 📊 Funcionalidades do Dashboard

* Filtros interativos (Segmentação de Dados / Slicers)
* Gráficos dinâmicos
* Indicadores (KPIs)
* Visualização por tipo de assinatura
* Comparação entre planos

---

## 🛠️ Ferramentas Utilizadas

* Microsoft Excel
* Tabelas Dinâmicas
* Segmentação de Dados (Slicers)
* Gráficos
